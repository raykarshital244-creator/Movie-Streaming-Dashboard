# 🎬 Movie Streaming Dashboard - Power BI

## 📌 Project Overview
This project is an interactive Power BI dashboard developed to analyze movie streaming data. The dashboard provides insights into movies, genres, ratings, streaming platforms, and audience trends using interactive visualizations.

## 🎯 Objectives
- Analyze movie streaming trends.
- Identify the most popular genres.
- Compare ratings across different movies.
- Monitor content distribution and performance.
- Provide interactive filtering for better analysis.

## 📊 Dashboard Features
- Total Movies KPI
- Average Rating KPI
- Genre-wise Analysis
- Platform-wise Distribution
- Rating Distribution
- Release Year Analysis
- Interactive Slicers and Filters

## 🛠️ Tools & Technologies
- Microsoft Power BI
- Power Query
- DAX
- Microsoft Excel / CSV Dataset

## 📁 Files Included
- `final report.pbix` – Power BI dashboard
- `README.md` – Project documentation
- `Dashboard.png` – Dashboard screenshot (optional)
- `Dataset.csv` – Sample dataset (optional)
